### 🚀 Featured Projects

#### 1. [Archon: The Intelligent Architect](https://huggingface.co/spaces/Prince-Alca-Archon)
A full-stack RAG application enabling private document ingestion and semantic search.
- Multi-session conversation memory powered by **ChromaDB**
- Built solo across Python, Flask, and JavaScript — containerized with **Docker**
- Live on **HuggingFace Spaces**

#### 2. [Multi-Agent Sales Analysis System](https://github.com/princeemmanuelalca-art)
A sophisticated 5-agent **LangGraph** pipeline for automated data science workflows.
- **Agents:** Orchestrator → Data Validator → Analyst → Code Executor → Report Writer
- Deterministic conditional routing + automated **Matplotlib** visualization generation

#### 3. [Self-Improving Coding Agent](https://github.com/princeemmanuelalca-art)
An autonomous agent that writes, executes, and debugs Python code in real-time.
- **Reflection loop:** agent analyzes execution errors and rewrites its own code
- Secured via **E2B cloud sandboxes** for safe untrusted code execution

#### 4. [AI Code Review Bot](https://github.com/princeemmanuelalca-art)
Automated professional-grade analysis for GitHub Pull Requests.
- Triggered via **GitHub Actions** + **PyGithub** on every PR
- Smart chunking handles large diffs within LLM token limits

#### 5. [Custom MCP Server (Python + Groq)](https://github.com/princeemmanuelalca-art)
A protocol-compliant server extending LLM capabilities via **Model Context Protocol**.
- Exposes live web search and URL summarization to MCP-compatible clients like Claude Desktop
- Powered by **Groq (LLaMA 3.3 70B)**
